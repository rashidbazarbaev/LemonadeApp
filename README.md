Вступление

Приложение Lemonade - это простое в использовании приложение, которое позволяет пользователю приготовить цифровой лимонад, выполнив ряд действий: сорвать лимон с дерева, выжать сок, выпить стакан лимонада и начать все сначала.

В этом приложении рассматриваются следующие концепции:

    Простая компоновка пользовательского интерфейса в Compose
    Добавление интерактивного поведения в composable
    Использование remember и mutableStateOf для отслеживания актуального состояния приложения
